# Data Science Project Structure

This folder shows a clean, reproducible layout for a data science project.

- data/raw: original data files (do not modify raw data)
- data/processed: cleaned or transformed data outputs
- notebooks: exploratory and analysis notebooks
- models: saved trained model artifacts
- outputs: generated results, tables, and visualizations

This structure separates code/data/outputs and makes the project easier to share and review.
